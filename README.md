# Tools Made Easy

Tools Made Easy is a simple web application that helps users find free online design tools in one place. It organizes commonly used tools like background removal, image compression, and image resizing into easy categories so users do not have to search across multiple websites.

## Features

- Categorized list of free design tools
- Search functionality to quickly find tools
- Simple and clean user interface
- Responsive layout for different screen sizes

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Idea

The idea behind this project is to reduce the time spent searching for reliable free design tools. Instead of visiting multiple websites, users can access everything from a single organized platform.
